# nestjs-typeorm-mongodb

## Project setup

```bash
$ bun install
```

## Compile and run the project

```bash
# development
$ bun start

# watch mode
$ bun start:dev

# production mode
$ bun start:prod
```

## Run tests

```bash
# unit tests
$ bun run test

# e2e tests
$ bun test:e2e

# test coverage
$ bun test:cov
```
